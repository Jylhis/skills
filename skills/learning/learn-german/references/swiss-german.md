# Swiss German (Schweizerdeutsch) for learners

## What it is — and what to target
- **Schweizerdeutsch is a family of Alemannic dialects spoken across
  German-speaking Switzerland** (Züridütsch, Bärndütsch, Baseldytsch, etc.). There
  is **no single standardized written form**.
- The **written language in Switzerland is Standard German (Swiss High German)** —
  newspapers, signs, schoolbooks, official documents. It differs from German
  Hochdeutsch only slightly (see below).
- Practical target: **learn Hochdeutsch for reading/writing/grammar; learn Swiss
  German as a spoken overlay** for understanding and everyday speech. Introduce it
  from ~A2 for listening, always flagged as dialect.

## Swiss *Standard* German vs German Hochdeutsch (written differences)
- **No ß** — Switzerland writes "ss" everywhere ("Strasse", "dass", "gross").
- Some lexical preferences: **Velo** (Fahrrad/bike), **Billett** (Ticket),
  **Glace** (Eis/ice cream), **Natel** (Handy/mobile), **parkieren** (parken),
  **grillieren** (grillen), **Trottoir** (Bürgersteig), **Poulet** (Hähnchen).
- Greetings: **Grüezi** (formal hello), **Grüezi mitenand** (to several),
  **Hoi/Sali** (informal), **Merci vielmal** (thanks a lot), **Adieu/Tschüss**.

## Spoken Schweizerdeutsch — high-value features to teach for listening
- **No Präteritum** — the simple past is essentially absent; the Perfekt is used
  for all past ("ich ha gmacht" = ich habe gemacht).
- **Diminutive -li** instead of -chen/-lein: Hüsli (Häuschen), Bürli, Müsli.
- **k → ch / x sound shift**: Kind → "Chind", Kuchen → "Chueche", sack → "Sack/Säckli".
- **-ig and vowel shifts**: nicht → "nöd/nid", ein → "es/e", klein → "chlii",
  Haus → "Huus", Zeit → "Ziit".
- **Numbers/greetings** differ enough to learn explicitly.
- Common words: **gäll?** (right?), **lueg** (schau/look), **öppis** (etwas),
  **nöd** (nicht), **gschpürsch?**, **es git** (es gibt).

## How to teach it
- Keep Swiss German in **listening/speaking** activities; keep writing in
  Hochdeutsch.
- When you give a Swiss word or phrase, show the Hochdeutsch equivalent beside it
  and tag the card `swiss-german` so it's clearly an overlay, not the standard.
- Note there's regional variation — pick one region (e.g. Zürich) as the default
  reference if the learner doesn't specify, and say so.
