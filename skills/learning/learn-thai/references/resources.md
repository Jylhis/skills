# Thai comprehensible-input and vocabulary resources

Pointers for level-appropriate input and frequency-first vocabulary. Confirm URLs
at use time.

## Frequency-first vocabulary
- A few hundred high-frequency words plus pronouns, classifiers, and common verbs
  carry most beginner conversation. Teach these before rare words.
- **Classifiers (ลักษณนาม)** are essential early — Thai counts nouns with a
  classifier (คน for people, ตัว for animals/clothes, อัน for small things,
  ใบ for sheets/containers). Teach the classifier with the noun.
- **Polite particles** ครับ (khráp, male) / ค่ะ-คะ (khâ/khá, female) shape nearly
  every sentence — teach them immediately.

## Listening / comprehensible input
- **Comprehensible Thai** (YouTube) — graded, picture-and-gesture input for
  beginners (pure comprehensible-input method).
- **ThaiPod101** — graded audio lessons with transcripts.
- **Learn Thai with Mod / Thai with Grace** — pronunciation and tone-focused video.
- Children's stories and song lyrics with known vocabulary make good early reading.

## Script and tone practice
- Build a consonant-class wall chart (mid/high/low) and review it until automatic.
- Use minimal-pair audio drills for the five tones (see tones.md) — perception
  before production.

## How to use these in a session
- Keep new vocabulary ~1–2 items per sentence; gloss inline (comprehensible input).
- For every new word, run `scripts/thai-translit.py` to confirm segmentation and
  tone, then make a card with Thai script + Paiboon + tone + classifier (for nouns).
- Drill perception (hear → pick the tone) before production (say the word).
- Reading: start with mid-class + long-vowel words, then add classes, finals, and
  tone marks (the sequence in script.md).
