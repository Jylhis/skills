# emacs-dev

Emacs and Elisp development intelligence for jstack: 6 skills covering
modern Elisp conventions, code review, ERT testing, debugging,
keybinding design, and Nix-based Emacs packaging.

## Contents

- `plugin.nix` — plugin manifest
- `skills/` — 6 skill directories with `SKILL.md`

This plugin is part of [jstack](../../) and is installed into
`~/.claude/plugins/emacs-dev/` automatically by `scripts/install.bash`.
There is no separate install step.

## Skills

`elisp-conventions`, `elisp-review`, `elisp-testing`, `emacs-debugging`,
`emacs-keybindings`, `emacs-nix-packaging`
