## Skills

### `session-log`

Summarizes the current Claude Code conversation and appends the result to
a weekly agent-log file named `YYYY-wWW agent-log.md` (ISO week, computed
via `date +%Y-w%V`).

**Trigger phrases:** "log this", "session log", "summarize this session",
or any request to write results to the agent-log.

**Format rules** (enforced by the skill):

- Reverse chronological order, newest day on top
- One `## YYYY-MM-DD` heading per day
- Plain bullets for topics, nested bullets for details
- `CHUNK: <title>` markers point at reusable outputs from the session
- No intros, no meta-commentary, append-only

**Cross-week handling:** If a topic belongs to a date in a different ISO
week than the target file, the skill prompts the user to choose which
file to write into.
