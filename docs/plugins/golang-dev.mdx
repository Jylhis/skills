`plugins/golang-dev/` ships 36 Go skills aggregated from
[`samber/cc-skills-golang`](https://github.com/samber/cc-skills-golang) (MIT,
by Samuel Berthe) and
[`JetBrains/go-modern-guidelines`](https://github.com/JetBrains/go-modern-guidelines)
(Apache-2.0).

<Note>
The plugin's README references `gopls` via `nix run nixpkgs#gopls`, but no
`.mcp.json` exists and `gopls` is not currently in `runtime/default.nix`. To
make `gopls` available to Claude Code, add it to the runtime.
</Note>

## Layout note

Many skills carry `references/`, `evals/`, and `assets/` subdirectories
with longer-form material. Only `SKILL.md` frontmatter is loaded into the
skill index at startup; everything else is on-demand reference content.
